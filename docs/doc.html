Implementace

Stručná dokumentace k implementaci, která popisuje, které části projektu (např. PHP skripty) implementují jednotlivé případy použití.

Pri tvorbe aplikácie sme použili architektonický vzor MVC - model, view, controller

Model vrstvu tvorili modely jednotlivých entít ERD v súboroch [name].php. Tieto súbory slúžili na tvorbu relácií medzi entitami, obsahovali metody na interakciu s DB a poskytovali rozhranie pre controlers.

Controller vrstvu tvorili controllers jednotlivých modelov v súboroch [name].php. Controllers slúžili na prijatie požiadavku pomocou prevolania metod rountingom. Controllers interagovali s modelmi a tvorili views,
ktoré následne poslali na klientskú stranu aplikácie. Tvorili tzv. business logiku aplikácie.

View vrstvu tvorili views jednotlivých podstránok aplikácie + komponenty aplikácie v súboroch [name].blade.php. Každé view interaguje s užívateľom pomocou GUI a následne prevoláva controllers, ktoré reagujú na jeho
požiadavky a vracajú odpoveď v podobe zmeny view alebo dát aplikácie.

Pri stylovaní front-endu sme použili SASS. Je to programovací jazyk, ktorý je vpodstate preprocessingom CSS, teda musí byť pre použitim skompilovaný kompilátorom, ktorý zo SASS vytvorí výsledný CSS súbor.
SASS nám dovoluje používať premenne, funkcie, konštanty, znovupoužitelný kód a hlavne modularitu zdrojového kódu.

Inštalácia

Projekt je možné spustiť, iba ak je dostupná celá štruktúra priečinkov. Projekt bude dostupný ako verejný repozitár na githube.

Prikazy na spustenie projektu na localhoste

composer install
cp .env.test .env
php artisan key:generate
npm install
php artisan serve

Vytvorenie lokálnej databázy

Úprava súboru env - connection string na databázu

Úprava súboru config/database.php - connection string na databázu

Nastavenie hodnoty APP_KEY v .env súbore - php artisan key:generate

Spustenie migrácii a seedovania dát - php artisan migrate:refresh --seed

Spustenie servera - php artisan serve

Pre editáciu css a js je potrebné nainštalovať npm.

npm install
