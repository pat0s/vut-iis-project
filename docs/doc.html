Inštalácia

Projekt je možné spustiť, iba ak je dostupná celá štruktúra priečinkov. Projekt bude dostupný ako verejný repozitár na githube. composer install cp .env.test .env php artisan key:generate npm install Vytvorenie lokálnej databázy Úprava súboru env - connection string na databázu Úprava súboru config/database.php - connection string na databázu Nastavenie hodnoty APP_KEY v .env súbore - php artisan key:generate Spustenie migrácii a seedovania dát - php artisan migrate:refresh --seed Spustenie servera - php artisan serve Pre editáciu css a js je potrebné nainštalovať npm. npm install
